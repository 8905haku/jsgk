## About
A lightweight JavaScript game framework for building 2D games with HTML5 Canvas.

## Features

- Game loop system
- Scene management
- Component-based game objects
- Simple and lightweight architecture

## Installation

Clone the repository:

```bash
git clone https://github.com/yourname/js-gamekit.git
```


## Example

```
index.html

<html>
  <head>
    ...
    
    <script type="module" src="./game/main.js"></script>

  </head>
  <body >

    <canvas id="screen"></canvas>
    
  </body>
</html>

```
